# Car Rent

Car Rent is a simple application for managing a car rental system.

## Table contains

- [Car brand]
- [Car model]
- [Year of manufacture]
- [Category (e.g. Economy, SUV, Luxury)]
- [Availability status]
- [Rental price]

## Features

- Add, edit, and delete cars.
- View all available cars for rent.
- Rent a car by selecting a specific period.
- Manage car availability status (available / rented).
- Search and filter cars by brand, model, or category.
- Simple and user-friendly interface.

## Using

- Java
- HTML / CSS
- Maven
- CRUD
